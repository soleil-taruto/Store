============
AddGradation
============

----
説明

画像ファイルに明暗のグラデーションを適用する。


----
コマンド

AddGradation.exe (/W | /B) 左上レベル 右上レベル 左下レベル 右下レベル (ファイル名 | フォルダ名)

	/W ... 白色マスク (明るい方へ傾倒)
	/B ... 黒色マスク (暗い方へ傾倒)

	各レベルには 0 〜 255 (弱〜強) の整数を指定すること。

	ファイル名を指定すると、そのファイルのみ処理する。
	フォルダ名を指定すると、そのフォルダの直下にある画像ファイル (.bmp .gif .jpg .jpeg .png) を処理する。

	出力先は C:\1〜999 である。


実行例

	AddGradation.exe /W 100 0 0 0 .\Image.png

	AddGradation.exe /B 0 0 0 70 C:\temp\Images
