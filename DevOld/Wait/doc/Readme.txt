====
Wait
====

----
説明

TIMEOUT n コマンドのように n 秒間待ちます。その間キー入力も受け付けます。


----
コマンド

Wait.exe WAIT-SEC

	WAIT-SEC ... 初期の残り時間を示す秒数
	             指定できる範囲：0 〜 999999


実行例

	Wait.exe 60
	Wait.exe 180
	Wait.exe 3600


----
キー入力

キー               結果
-----------------------------------------------
エスケープキー   直ちに終了する。エラーレベル 1
エンターキー     直ちに終了する。エラーレベル 0
+ キー           残り時間 + 1 min , 但し残り時間が 999999 sec を超える場合は、残り時間を 999999 sec にする。
- キー           残り時間 - 1 min , 但し残り時間が 3 sec 未満になる場合は、残り時間を 3 sec にする。
* キー           残り時間 + 1 hour
/ キー           残り時間を 3 sec にする。


★エラー終了時のエラーレベルは 2 です。
